# reverse-shift-cipher
This is a custom symmetric text encryption project built with Python.

The program encrypts a message by reversing the text and shifting each letter by a user-provided numeric key. The same key is used to decrypt the message by shifting the letters back and reversing the text again.

## Features

- Encrypts text using a custom reverse-shift algorithm
- Decrypts text using the same numeric key
- Preserves uppercase and lowercase letters
- Keeps spaces, numbers, and punctuation unchanged
- Saves encrypted output to a text file
